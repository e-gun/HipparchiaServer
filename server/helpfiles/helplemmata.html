Searching by lemmata can be enabled by clicking on the box next to λ once complex searching is selected.

A box that has its λ checked will only accept a word that has been lemmatized. Lemmata are lists of known forms of a word. Not every possible form is represented. If there was no dative plural of some word found when the original data was compiled (ages ago and not necessarily with the same data in use here and now...), then that word is not a part of the word list.

Some warnings:
  • If you defy the autocomplete suggestions and enter a dictionary form that is not known, expect to get 0 results.
  • βολε͂ϲ and ⒣όρϙοϲ will not be found if you search for "all forms of βουλή" or "all forms of ὅρκοϲ".
  • Since this search is sensitive to accents, enclitics like μοι that change accentuation will produce misses.
  • Some word lists are huge. 128 exact froms can be mighty slow, especially if this is a common verb.
Consider double-checking anything desperately important via REGEX searches.
